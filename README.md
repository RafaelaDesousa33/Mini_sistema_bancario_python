Mini Sistema Bancário em Python

Este projeto é um sistema bancário simples desenvolvido em Python, permitindo ao usuário realizar operações como depósito, saque e consulta de saldo de maneira prática pelo terminal.

- Funcionalidades

Criar conta bancária (conforme estrutura do código)

Depositar valores

Sacar valores com verificação de saldo

Exibir saldo atual

Mostrar extrato ou histórico de movimentações

Menu interativo no terminal

- Tecnologias Utilizadas

Python 3

Recursos da biblioteca padrão:

Entrada e saída de dados

Condicionais

Estruturas de repetição

Manipulação de variáveis

- Como executar o projeto

Certifique-se de ter o Python 3 instalado.
Verifique com:

python --version


Faça o clone do repositório:

git clone https://github.com/RafaelaDesousa33/Mini_sistema_bancario_python


Acesse a pasta:

cd Mini_sistema_bancario_python


Execute o script principal (ajuste o nome se necessário):

python main.py

- Exemplo de Uso

O menu apresenta opções como:

=== Sistema Bancário ===
[1] Depositar
[2] Sacar
[3] Ver Extrato
[0] Sair


E o sistema responde conforme a operação escolhida.
